# 你好，Pelican！

这是我用 Pelican 生成的第一篇文章。

Pelican 是一个 **Python** 静态站点生成器。

## 特点
- 纯静态，加载快
- 不需要数据库
- 基于 Python

> 静态站点，动态人生。

你可以在文章中写 `行内代码` 或代码块：

```python
def hello():
    print("Hello, Pelican!")
```
